# shared

Базовые переиспользуемые части: UI-кирпичики, утилиты, типы, конфиг, клиенты API.

Рекомендуемая структура:
- `shared/ui` — примитивы интерфейса
- `shared/lib` — утилиты/хелперы
- `shared/api` — базовые клиенты/инстансы
- `shared/config` — конфиг приложения
- `shared/types` — общие типы
