# FSD (Feature-Sliced Design)

Слои (верх → низ):
- `app` — точки входа/роутинг Next.js (App Router)
- `processes` — сквозные процессы (онбординг, авторизация и т.п.)
- `pages-layer` — композиция страницы из виджетов (переименовано, чтобы не конфликтовать с Next.js `pages` router)
- `widgets` — крупные блоки UI
- `features` — пользовательские фичи (действия)
- `entities` — доменные сущности
- `shared` — переиспользуемая база

Импорты:
- Используй `@/...` для импортов из `src`.
